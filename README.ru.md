# Aurora Browser

Настольный браузер на Electron с красивым UI в стиле macOS и поддержкой тёмной темы.

[English](./README.md) | [Русский](./README.ru.md)

## Возможности

- 🌙 **Тёмная тема по умолчанию** - Красивый тёмный интерфейс сразу после запуска
- 🎨 **UI в стиле macOS** - Современный дизайн с вкладками, адресной строкой и навигацией
- ⚡ **Быстрый и лёгкий** - Построен на Electron для кроссплатформенной совместимости
- 🔒 **Безопасность** - Песочница с стандартной безопасностью браузера

## Установка

### Установщик
1. Скачайте `Aurora Browser Setup 1.0.0.exe`
2. Запустите установщик
3. Запустите Aurora Browser

### Портативная версия
1. Скачайте `Aurora Browser 1.0.0.exe`
2. Запустите напрямую - установка не требуется

## Разработка

### Требования
- Windows 10/11
- Node.js LTS

### Установка
```bash
git clone https://github.com/Allex863/aurora-browser.git
cd aurora-browser
npm install
npm start
```

### Горячие клавиши
- `Ctrl+L` - Фокус на адресную строку
- `Ctrl+T` - Новая вкладка
- `Ctrl+W` - Закрыть вкладку
- `Ctrl+R` - Обновить

### Сборка
```bash
npm run dist
```
